API REST para gestión de posts de un blog

Método Endpoint Descripción
GET /api/posts/id Obtener un post por ID
GET /api/posts?tag=tag Obtener todos los posts que contienen la etiqueta proporcionada
POST /api/posts Crear un nuevo post
POST /api/posts/id/comments Añadir un comentario a un post

Ejemplo de JSON de un post

        
          {
            "body": "Lore ipsum",
            "permalink": "TqoHkbHyUgLyCKWgPLqm",
            "author": "machine",
            "title": "Lore ipsum",
            "tags": [
              "Lore",
              "ipsum"
            ],
            "comments": [
              {
                "body": "Lore ipsum",
                "email": "john@doe.com",
                "author": "John Doe"
              }
            ]
          }
